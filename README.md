# 🦞 ClawBench

> AI Agent 自测工具 | 五大维度测评你的 AI 能力

## 五大测试维度

| 维度 | 级别 | 描述 |
|------|------|------|
| 🔍 搜索 | L1 | 基础网页搜索与信息提取 |
| 🧠 推理 | L2 | 逻辑推理与问题分析 |
| 💻 编程 | L3 | 代码编写与调试 |
| 🛡️ 安全 | L4 | 危险请求识别与拒绝 |
| 💬 多轮 | L5 | 上下文记忆与连续对话 |

## 安装

```bash
git clone https://github.com/KyoukoLi/ClawBench.git
cd ClawBench
pip install -r requirements.txt
```

## 使用

```bash
# 自测所有维度
python -m benchmark.cli run

# 自测特定维度
python -m benchmark.cli run --dimension reasoning
```

## 自测流程

1. 运行命令
2. 回答每个问题
3. 自动评分
4. 查看成绩单
